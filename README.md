# BBC Navigation Test (Playwright + Cucumber)

## Tech Stack
- Playwright
- TypeScript
- Cucumber (BDD)

## Test Scope
Verify BBC News navigation menu works in:
- Desktop view
- Mobile view

## Features
- Page Object Model
- Parallel execution
- Thread-safe browser contexts
- Responsive testing (viewport + device emulation)

## Install

npm install

## Run Tests

npm test
